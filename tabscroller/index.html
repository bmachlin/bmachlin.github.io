Tab Scroller

Use the format #[MM:SS pp t] to identify a timestamp (position and/or type optional). Time 00:00 not allowed
p = position [##] => % of screen height to scroll to at this time. 99 means bottom, 00 means top, default is 50
t = type [s|n] => Scroll (default), or sNap to this pos at this time
