Musique

A senior capstone project that implemented democratic music queing. Users could host and join "parties" in which partygoers could queue songs and vote on songs in the queue. The queue would automatically sort by number of votes and playback through the host's device. We used the Spotify API and Firebase to create a functional Android app.
